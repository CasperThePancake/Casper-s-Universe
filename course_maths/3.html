Exponential Functions

You've probably heard of exponential growth before. It's growth that keeps becoming faster, because the speed of the growth depends on the input value. An example function would be y=2x.

In general, we'll say the equation of an exponential function is f(x)=ax. We'll also have to account for some stuff here. Our value for a cannot be negative, since we can't always calculate a negative number to any power. For example, a negative number to the power of 1/2 is the square root of a negative number, which doesn't exist. This means that a must be positive. It also can't be equal to 1. That's not because that wouldn't exist, but that would just result in a constant function (y=1). Additionally, we can add b into the mix and say the equation is f(x)=b*ax.

Properties

If a is somewhere between 0 and 1, we get a function like this:

In general, the domain of this function is every real number.
The range is every strictly positive real number (so larger than 0). This is because any positive number to a certain power will result in a positive number.
This function has no zeros.
It has one intersection with the y-axis, namely (0,1). If you add a b in front of the equation, this point will always be (0,b).
The x-axis acts as a horizontal asymptote; the farther to + or - infinity we go, the closer the function value gets to it, but it never quite reaches it completely.
If a>1, f is increasing for the entire domain. If 0<a<1, f is decreasing for the entire domain.

Logarithms

Definition

Look at this:$$\log_{2}(8)$$The meaning of this new thing is as follows: "To what power must we raise 2 in order to get 8?" This logarithm would thus equal 3. Below are a few more examples to get you acquainted with this new maths tool:$$\log_{16}(4)=1/2$$ $$\log_{7}(7)=1$$ $$\log_{69}(1)=0$$ $$\log_{1}(5)=???$$ $$\log_{-4}(16)=???$$Those last two are not possible. You might think the answer to the last one is 2, but we cannot use negative numbers inside of our logarithm. The same goes for 1, since you can't get anything else than 1 if you raise it to any power.

a^(log(a)(x))=x

This is a special little trick:$$a^{\log_{a}(b)}=b$$Since we don't know the answer to the logarithm in the exponent, we'll call it y. This means that log(a)(b)=y, or a^y=b. If we replace the logarithm by what we defined it as (y), we get a^y, which we just said is equal to b. So the entire thing equals b. Cool, right?

Difference of logarithms

Next up we will prove the following:$$\log_{a}(\frac{x_{1}}{x_{2}})=\log_{a}(x_{1})-\log_{a}(x_{2})$$The proof for this is completely analogue to the previous one, with the only difference being that the division causes that exponent to be y1-y2 instead of y1+y2.

Changing the base of a logarithm

The base of a logarithm is the first number, so in log2(4), the number 2 is the base of the logarithm. Logarithmic functions

Finally let us analyze the graph of a logarithmic function. The logarithmic function is the inverse function of the exponential function, which means that if we mirror the exponential function over y=x, we get a logarithmic function.

The red graph is y=2x.
The blue graph is y=x.
The green graph is y=log2(x).

Properties

Since this function is the inverse of the exponential function, its domain and range are swapped. The range is every real number, while the domain is only strictly positive numbers.
While the exponential function had no zeros and one intersection with the y-axis, the logarithmic function has one zero (1,0) and no intersections with the y-axis.
The exponential function had a horizontal asymptote of y=0 (x-axis), while the logarithmic function has a vertical asymptote of x=0 (y-axis).

Additionally, the value of a in loga(x) says something about the global increase or decrease. In the graph above, a>1, which causes the graph to increase. In the graph below, 0<a<1, which causes the graph to decrease.
